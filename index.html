Projects

Row Counter Website

The Row Counter Website is a user-friendly web application that I developed to assist knitters and crocheters in keeping track of their stitches and rows while working on their projects.

The website features a clean and intuitive interface, allowing users to easily input the number of rows or stitches they want to track. The counter automatically increments or decrements based on the user's input, making it effortless to keep track of progress.

I built the Row Counter Website using HTML, CSS, and JavaScript. The project involved implementing event handling and dynamic DOM manipulation to create a seamless user experience. Additionally, the website is responsive, ensuring it works flawlessly on both desktop and mobile devices.

This project not only enhanced my web development skills but also provided me with the opportunity to address a real-world problem faced by knitting and crocheting enthusiasts. It has been well-received by users in the crafting community and has motivated me to continue exploring web development as a means to solve practical challenges.
